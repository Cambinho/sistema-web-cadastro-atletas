# 📚 Sistema de Gerenciamento de Biblioteca

Este projeto é um sistema de gerenciamento de biblioteca desenvolvido em Python, utilizando Programação Orientada a Objetos e arquitetura MVC (Model-View-Controller), com integração a banco de dados.

O sistema permite o controle de usuários, livros e empréstimos, simulando o funcionamento de uma biblioteca real.

---

## 🚀 Funcionalidades

- Cadastro de usuários
- Cadastro de livros
- Realização de empréstimos
- Controle de devoluções
- Listagem de usuários
- Listagem de livros
- Listagem de empréstimos
- Validações de regras de negócio

---

## 🛠️ Tecnologias utilizadas

- Python
- SQLite / PostgreSQL
- Programação Orientada a Objetos (POO)
- Arquitetura MVC

---

## 📂 Estrutura do projeto


projeto_biblioteca/
│
├── models/
├── views/
├── controllers/
├── database/
├── main.py


O projeto segue o padrão MVC:
- **Models** → regras de negócio e estrutura dos dados
- **Views** → interface com o usuário
- **Controllers** → controle das operações do sistema

---

## ▶️ Como executar o projeto

1. Instalar o Python 3
2. Clonar este repositório:

git clone https://github.com/Cambinho/sistema-biblioteca-python

3. Acessar a pasta do projeto:

cd sistema-biblioteca-python

4. Executar o sistema:

python main.py
